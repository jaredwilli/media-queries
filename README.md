# Media Queries

This repo contains media queries for all standard devices, which include:

 - Phones and Handhels
   - iPhones
   - Galaxy Phones
   - HTC Phones
 - Tablets
   - iPads
   - Galaxy Tablets
   - Nexus Tablets
   - Kindle Fire
 - Laptops
 - Wearables
   - Apple Watch
   - Moto 360 Watch

Source: [CSS-Tricks](https://css-tricks.com/snippets/css/media-queries-for-standard-devices/)
